# Changelog

All notable changes to this project will be documented in this file.

The format is based on Keep a Changelog and this project follows semantic-style release tags even though the product is still early.

## [0.1.0] - 2026-03-25

### Added

- Repositioned the project as `ClipScript`, a subtitle-first short-form clip generator for creators
- Added a built-in demo flow so contributors and evaluators can inspect the product without making an API call
- Added screenshots, social assets, CI, contribution guides, issue templates, and release documentation
- Added Docker support for local deployment
- Added source-video upload and `ffmpeg`-based clip export flow

### Changed

- Simplified the public product pitch around creator workflows and editor handoff
- Clarified how OpenAI-compatible APIs are configured in the UI

### Fixed

- JSON parsing errors in AI responses
- TypeScript and ESLint configuration issues
- Security-related dependency updates for the web stack
