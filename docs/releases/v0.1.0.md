# v0.1.0

## Highlights

- Repositioned the project as `ClipScript`, a subtitle-first short-form clip generator
- Added a built-in demo flow so the repo is easier to evaluate quickly
- Redesigned the home and preview experience for a clearer public-facing product pitch
- Added Open Graph assets, screenshots, CI, contributing guides, and launch documentation
- Cleaned up the API and video export messaging around creator workflows

## What you can do

- Upload an `.srt` file and generate clip ideas with AI
- Review titles, hooks, captions, voiceover angles, and editing direction
- Export a CSV execution table for editors
- Upload the source video and cut the selected ranges with `ffmpeg`

## Next

- Add more transcript formats
- Improve preview and approval tooling
- Ship stronger batch workflows for creator libraries
