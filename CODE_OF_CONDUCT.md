# Code of Conduct

## Our Standard

This project aims to be useful, direct, and respectful. We welcome contributions, questions, and critiques as long as they stay constructive and technically grounded.

Examples of behavior that contribute to a positive environment:

- Giving clear, actionable feedback
- Assuming good intent while discussing code or product decisions
- Respecting different levels of experience
- Focusing on the problem, not the person

Examples of unacceptable behavior:

- Harassment, insults, or personal attacks
- Discriminatory language or exclusionary behavior
- Deliberate disruption, spam, or bad-faith arguments
- Publishing private information without permission

## Scope

This Code of Conduct applies to project discussions, issues, pull requests, and any community spaces linked from this repository.

## Enforcement

Project maintainers may remove comments, close threads, or block contributors whose behavior violates this Code of Conduct.

## Reporting

If you encounter a serious issue, open a private report via the repository owner contact listed on GitHub.
